# PUBG 매칭 시스템은 정말로 공정한가?

## 매칭 시스템이란?

**매칭 시스템(Matchmaking System)**: 온라인 게임에서 플레이어들을 자동으로 매칭시켜 게임을 진행할 수 있도록 하는 시스템

### 매칭 시스템의 목표
- **공정성**: 비슷한 실력의 플레이어끼리 매칭하여 균형 잡힌 경기 제공
- **효율성**: 빠른 매칭 시간으로 대기 시간 최소화

### 일반적인 매칭 방식
- **실력 기반 매칭(SBMM - Skill-Based Matchmaking)**: 플레이어의 실력 점수나 랭크를 기준으로 매칭
- **무작위 매칭**: 실력과 상관없이 무작위로 플레이어들을 배치
- **혼합 방식**: 실력과 대기 시간을 모두 고려한 절충안

### 매칭 시스템의 중요성
- 플레이어간 격차가 크면 게임에 충분한 재미를 느낄 수 없어 유저 이탈로 이어질 수 있다.
- 매칭 대기 시간이 너무 길어져도 플레이어가 게임 한 판을 하기 위해 많은 시간을 소비해야하므로 게임을 하고 싶은 마음이 사라진다.

## PUBG (PlayerUnknown's Battlegrounds) 게임 소개

### 게임 개요
**PUBG**는 100명의 플레이어가 하나의 맵에서 생존을 위해 경쟁하는 **배틀로얄** 장르의 대표 게임. 게임이 시작되면 모든 플레이어가 비행기에서 낙하산으로 맵에 착륙하여, 무기와 장비를 수집하며 마지막 1명(또는 1팀)이 남을 때까지 싸우는 것이 목표이다.

### 핵심 게임 메커니즘
- **플레이어 수**: 최대 **100명**이 한 게임에 참여
- **맵 축소**: 시간이 지나면서 **안전지대**가 점점 줄어들어 플레이어들을 강제로 접근시킴
- **순위 시스템**: 최종 생존 순서에 따라 1위부터 100위까지 순위가 결정됨
- **장비 수집**: 맵 곳곳에 있는 무기, 방어구, 회복 아이템 등을 획득해야 함

### 매칭 시스템 특징
기본적으로 솔로는 솔로끼리만, 듀오는 듀오끼리만, 스쿼드는 스쿼드끼리만 매칭됩니다. 각 모드는 완전히 분리되어 운영되며, 모드 간 혼재 매칭은 발생하지 않습니다.

## 매칭 모드별 특징

### 솔로 모드 (Solo)
- **구성**: 1명씩 100명이 개별적으로 경쟁
- **특징**: 순수한 개인 실력으로만 승부
- **목표**: 마지막 1명까지 생존

### 듀오 모드 (Duo)
- **구성**: 2명씩 팀을 이뤄 총 50개 팀(100명)이 경쟁
- **특징**: 팀원과의 협력, 의사소통, 전략적 플레이 가능
- **목표**: 마지막 1개 팀까지 생존

### 스쿼드 모드 (Squad)
- **구성**: 4명씩 팀을 이뤄 총 25개 팀(100명)이 경쟁
- **특징**: 더 복잡한 팀워크와 역할 분담 필요
- **목표**: 마지막 1개 팀까지 생존

## 연구 문제 및 가설

### 핵심 연구 질문
**메인 질문**: PUBG의 솔로, 듀오, 스쿼드 모드 중 어떤 모드가 가장 공정한 매칭을 제공하는가?

**세부 질문**:
1. **모드별 실력 분산**: 각 모드에서 같은 매치 내 플레이어들의 실력 편차는 얼마나 다른가?
2. **매칭 품질 비교**: 솔로/듀오/스쿼드 중 어떤 모드가 가장 균등한 매칭을 제공하는가?
3. **팀 구성의 영향**: 듀오와 스쿼드에서 팀 내 실력 차이가 전체 매칭 공정성에 미치는 영향은?

### 가설 설정
- **H1**: 솔로 모드가 듀오/스쿼드보다 더 공정한 매칭을 제공할 것이다 (팀 변수가 없으므로)
- **H2**: 듀오/스쿼드에서 팀원 간 실력 차이가 클수록 전체 매칭의 공정성이 떨어질 것이다
- **H3**: 스쿼드가 듀오보다 팀 내 실력 분산의 영향을 더 크게 받아 공정성이 낮을 것이다

## 데이터셋

### Kaggle Dataset 사용
- **출처**: [PUBG Match Deaths](https://www.kaggle.com/datasets/skihikingkevin/pubg-match-deaths)

### 주요 Attributes
- **date**: 게임 진행 날짜
- **match_id**: 매치 고유 식별자
- **match_mode**: 게임 모드 (solo, duo, squad)
- **party_size**: 파티 크기 (1=솔로, 2=듀오, 4=스쿼드)
- **player_kills**: 킬 수
- **player_dmg**: 총 데미지
- **player_survive_time**: 생존 시간 (초)
- **player_dist_ride**: 차량 이동 거리
- **player_dist_walk**: 도보 이동 거리
- **player_assists**: 플레이어 어시스트 수
- **team_id**: 팀 식별자
- **team_placement**: 팀 최종 순위 (1위~100위)

# 매칭 공정성 분석 방법론

## Phase 1: 플레이어 실력 측정 및 분류

### 1.1 실력 점수 산출 - 가중평균 + Linear Regression 검증

**목적**: 다양한 게임 지표를 종합한 단일 실력 점수 생성

**방법**:
1. **가중평균 계산**: 킬 수, 데미지, 생존 시간, 이동 거리, 어시스트를 가중치를 적용하여 실력 점수 산출
2. **Linear Regression 검증**: 실력 점수 → 게임 순위 예측으로 점수의 유효성 검증

**출력**: 각 플레이어별 표준화된 실력 점수 (0-100 스케일)

### 1.2 실력 그룹 분류 - Percentile 기반 + KNN

**목적**: 플레이어를 5개 실력 그룹으로 분류

**방법**:
1. **초기 그룹 생성**: 실력 점수를 기준으로 5분위(20%씩) 그룹 생성
   - 그룹 1: 하위 0-20% (초급)
   - 그룹 2: 20-40% (하급) 
   - 그룹 3: 40-60% (중급)
   - 그룹 4: 60-80% (상급)
   - 그룹 5: 상위 80-100% (고급)

2. **KNN 분류**: 새로운 플레이어의 실력 그룹을 KNN으로 예측

**활용**: 각 매치에서 실력 그룹별 분포 균등성 측정

---

## Phase 2: 모드별 매칭 공정성 분석

### 2.1 솔로 모드 공정성 분석 - Logistic Regression

**목적**: 솔로 매치가 공정한지 불공정한지 이진 분류

**입력 특성**:
- 매치 내 실력 분산 (표준편차)
- 최고-최저 실력 차이 
- 평균 실력 수준
- 실력 그룹별 분포 균등성

**출력**: 매칭 공정성 (공정=1, 불공정=0)

### 2.2 듀오 모드 공정성 분석 - SVM

**목적**: 듀오 매치의 복잡한 공정성 패턴 분석

**입력 특성**:
- 팀 내 실력 차이 (팀원 간 실력 점수 차이)
- 팀 간 평균 실력 차이
- 전체 매치 실력 분산
- 팀 구성 밸런스 지수

**활용**: 비선형 커널을 사용하여 복잡한 팀 구성 변수들 간의 관계 분석

### 2.3 스쿼드 모드 공정성 분석 - Random Forest

**목적**: 스쿼드의 다차원적 팀 구성 요소들이 공정성에 미치는 영향 분석

**입력 특성**:
- 팀 내 최대 실력 차이
- 팀 내 실력 분산
- 팀 간 평균 실력 균등성
- 역할 분배 균형도 (킬러, 서포터 등)

**장점**: 여러 의사결정나무로 복잡한 팀 구성 패턴을 포착하고 특성 중요도 분석 가능

---

## Phase 3: 모드 간 공정성 비교

### 3.1 종합 공정성 예측 모델 - MLP (다층 퍼셉트론)

**목적**: 모든 모드의 데이터를 동일한 기준으로 평가하여 공정성 비교

**입력**:
- 모드 타입 (one-hot encoding: solo, duo, squad)
- 실력 분산 지표들
- 팀 구성 복잡도
- 매치 특성 변수들

**출력**: 매칭 공정성 종합 점수 (0-100)

**활용**: 모드별 공정성을 동일한 척도로 비교 평가

### 3.2 모드별 공정성 순위 도출

**방법**: MLP 모델의 예측 결과를 바탕으로 모드별 평균 공정성 점수 계산 및 순위 결정

**목표**: 솔로, 듀오, 스쿼드 중 가장 공정한 모드 순위 도출

## 분석 방법론 요약

### 사용 모델별 역할
- **Linear Regression**: 실력 점수 유효성 검증
- **KNN**: 실력 그룹 분류  
- **Logistic Regression**: 솔로 모드 공정성 이진 분류
- **SVM**: 듀오 모드 비선형 패턴 분석
- **Random Forest**: 스쿼드 모드 복합 요인 분석 및 특성 중요도
- **MLP**: 모드 간 종합 비교 평가

### 예상 결과
- 각 모드별 공정성 점수 및 순위 (예: 솔로 85점, 듀오 78점, 스쿼드 72점)
- 모드별 주요 공정성 영향 요인
- 팀 구성이 공정성에 미치는 정량적 영향

### 1단계: 데이터 전처리 및 실력 측정
- Kaggle 데이터셋 정제 및 탐색적 데이터 분석
- 가중평균으로 실력 점수 산출 및 Linear Regression으로 검증

### 3단계: 종합 비교 분석
- MLP로 모드 간 공정성 통합 비교

- 각 모드별 공정성 특성 및 개선점 제시
